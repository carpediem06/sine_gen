# sine_gen

To generate a sine waveform (.wav) file.
Forked from https://github.com/moutend/getSineWave

## Compile

    $ ./go


## Usage / Options

* c : channels     			[nb]   			mono
* b : sample resolution [bits] 			16 bits
* r : sampling rate 		[Hz] 	 	  	44100 Hz
* f : frequency 				[Hz]     		440 Hz
* s : duration 					[sec]       5 s
* h : help
